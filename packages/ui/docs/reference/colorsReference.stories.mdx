# Colors Reference

One UI exports a few color palettes and color functions to assist with creating themes. Additionally, color palettes can be used to accomodate special circumstances that don't warrant adding a new color to the theme. For example, an email application might allow users to assign a color to each label, but the application shouldn't necessarily include the user-facing color options as part of its theme.

## `createSwatch(hexValue) => swatch`

Create a new 10 step swatch based on the provided hex.

##### Arguments

1. `hex: string` - A hex value to use as the basis for the swatch. It should be near the desired middle of your swatch.

##### Returns

`swatch: object` - A swatch with shades `50-900`. Also includes a `closest` prop to denote the input color's position on the swatch.

##### Examples

```javascript
import { createSwatch } from 'one-ui'

const indigo = createSwatch('#0e24c4')

const light = swatch[100]
const main = swatch[swatch.closest]
const dark = swatch[800]
```
