# Installation

You can get started with One UI in two steps:


## 1. Install Packages

Inside your React project directory, install One UI by running either of the following commands:

```
yarn add @one-dev/ui lodash prop-types react react-dom styled-components
```

```
npm i @one-dev/ui lodash prop-types react react-dom styled-components
```

## 2. Set Up Your App

For One UI to work correctly, you need to set up the [ThemeProvider](/?path=/docs/components-themeprovider--basic) at the root of your application. You should also add [CssBaseline](/?path=/docs/components-cssbaseline--basic) to normalize browser styles in your own components.

```javascript
// src/index.js

import { CssBaseline, ThemeProvider } from '@one-dev/ui'
import React from 'react'
import ReactDOM from 'react-dom'
import App from './components/App'

const rootElement = document.getElementById('root')
const rootComponent = (
  <ThemeProvider>
    <CssBaseline />
    <App />
  </ThemeProvider>
)

ReactDOM.render(rootComponent, rootElement)
```
